# 固件升级

## 接口地址

> POST /firmware

## 请求地址

| 参数名称 |      是否必须     |     说明 |
| ----------- |:---------------:| -----:|
| version     |       是        |       打印机版本信息 |
| sn          |       是        |       打印机唯一编号 |
| user        |       是        |       使用者 |
| model       |       是        |       固件模板 |

## 返回示例
```    
    TSCDA200.v1.1.2.user1.bin
```   
    
# 设备列表

## 接口地址

> GET /org/{id}/devices

## 请求参数

| 参数名称   |    是否必须     |     说明 |
| ----------- |:---------------:| -----:|
| id          |       是        |       公司ID |
| pre_pag     |       否        |       每页展示的打印机数量,默认20页 |
