SONMA API 文档
===================

`SONMA API` 是胜马云打印提供给第三方的API接入方式。

使用 `SONMA API` 可以快速接入胜马云打印功能。

如有任何对接疑问，请发邮件至 <openapi@sonma.com.cn>
